# Next 45 Mars Rover

Mars Rover Kata

## Node.js (Install)

Requirements:

- Node.js
- npm (Node.js package manager)
- Jest
- Enzyme

# Run the project

```bash
git clone https://github.com/Siyanda-Mzam/Next45-Mars-Rover
cd Next45-Mars-Rover
npm i
npm start
```

# Run the tests

```bash
npm test
```
